# OBSIDIAN — Password Manager

A secure, private and fully offline password manager built with pure web technologies.

---

## <img src="https://cdn.jsdelivr.net/npm/bootstrap-icons@1.11.3/icons/shield-lock.svg" width="20"/> Overview

OBSIDIAN is a personal password manager designed to work as a native application on any device — mobile or desktop — without relying on external servers or subscriptions.

All data is encrypted locally using **AES-256-GCM**, the same standard used by banks and governments. The master password is never stored anywhere — only you know it.

---

## <img src="https://cdn.jsdelivr.net/npm/bootstrap-icons@1.11.3/icons/lock.svg" width="20"/> Security

- AES-256-GCM encryption  
- PBKDF2 with 310,000 iterations  
- Master password never stored  
- Fully local encryption (zero server dependency)  
- Auto-lock with customizable timeout  
- Confirmation required for critical actions  

---

## <img src="https://cdn.jsdelivr.net/npm/bootstrap-icons@1.11.3/icons/database-lock.svg" width="20"/> Vault System

- Store credentials (site, user, password, URL, notes)  
- Multiple independent vaults  
- Password history (last 10 changes)  
- Real-time search and filtering  
- Categories system (Social, Work, Bank, etc.)  
- One-click copy to clipboard  

---

## <img src="https://cdn.jsdelivr.net/npm/bootstrap-icons@1.11.3/icons/exclamation-triangle.svg" width="20"/> Security Alerts

- Detects reused passwords  
- Alerts for outdated passwords (+90 days)  
- Visual prioritization (color-coded alerts)  
- Active alerts counter

## <img src="https://cdn.jsdelivr.net/npm/bootstrap-icons@1.11.3/icons/key.svg" width="20"/> Password Generator

- Length from 8 to 64 characters  
- Uppercase, lowercase, numbers, symbols  
- Exclude ambiguous characters  
- Real-time strength indicator (7 levels)
